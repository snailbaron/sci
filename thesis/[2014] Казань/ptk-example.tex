Это пример статьи для сборника трудов конференции, подготовленной с помощью стилевого файла
\verb$ptk.sty$. Пакет \verb$ptk.sty$ был подготовлен на основе стилевого пакета \verb$mmro.sty$,

(см. \verb$http://www.ccas.ru/voron/download/mmrosty.zip$).

Статья может содержать разделы 1-го уровня --- создаются с помощью команды \verb$\section$ ---
и подразделы 2-го уровня --- создаются с помощью команды \verb$\paragraph$.

\section{Название раздела}

Теоремы, леммы, утверждения оформляются с помощью стандартных окружений:
\begin{theorem}
Формулировка теоремы.
\end{theorem}
\begin{proof}
Выключные формулы без номеров оформляются так:
\[
\left(\frac{a+b\sqrt{5}}{2}\right)\left(\frac{a-b\sqrt{5}}{2}\right) = \frac{a^2-5b^2}{4}.
\]

А с номерами вот так:
\begin{equation}
(a+b)^n = \sum_{k=0}^n {n\choose k} a^{n-k} b^k. \label{f_example}
\end{equation}

Если формула пронумерована, как, например, (\ref{f_example}),
на нее должна быть ссылка в тексте.
\end{proof}

\paragraph{Подраздел.}
Источники в библиографии должны идти в порядке появления ссылок на них.
Ссылки вставляются автоматически. Например, \cite{bibBook,bibArticle,bibArticle2}.

О других возможностях по оформлению можно прочитать в документации к пакету \verb$mmro.sty$.

Работа выполнена при поддержке РФФИ, проект \No\,00-00-00000.
